Project Five: Managing the savn.local Domain

Project Description

Managed the savn.local Active Directory domain as part of a system administration exercise. Tasks included creating organizational units (OUs), generating local and domain user accounts, assigning users to OUs, and managing role-based access. Explored domain login behavior under normal and offline domain controller conditions. Emphasized proper VM management, administrative permissions, and adherence to enterprise-level best practices for user account security and system reliability. Delivered all required screenshots documenting each step, demonstrating practical Active Directory administration skills.

Skills Learned

  • Active Directory management, including creating OUs and managing user accounts
  • User account provisioning and domain vs local account distinctions
  • Assigning users to OUs and applying role-based access control
  • Troubleshooting domain login issues and understanding domain controller dependencies
  • VM management and safe shutdown procedures for enterprise systems
  • Microsoft Management Console (MMC) usage and Local Users & Groups snap-in
  • Capturing and documenting system administration tasks with screenshots
  • Applying best practices in system security and administrative operations
